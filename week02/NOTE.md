## 编程语言通识

### 语言按语法分类

+ 非形式语言
  - 中文
  - 英文
+ 形式语言（乔姆斯基谱系）
  - 0型: 无限制文法 ```?::=?```
  - 1型: 上下文相关文法 ```?<A>?::=?<B>?```
  - 2型: 上下文无关文法 ```<A>::=?```
  - 3型: 正则文法 ```<A>::=<A>?```


### 形式语言产生式

+ BNF
  - 语法结构
  - 符号
+ EBNF
+ ABNF


### 图灵完备

+ 命令式(图灵机)
  - goto
  - if 和 while
+ 声明式 
  - lambda
  - 递归

### 类型系统

+ 动态静态
+ 强类型弱类型
+ 符合类型
+ 子类型
